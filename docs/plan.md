# Plan

## Goal
做一个 Codex 菜单栏应用，用于本地账号切换和额度查看。

## Phase
MVP 设计与实现。

## Scope
- 菜单栏入口
- 账号列表
- 本地账号切换
- 当前账号 `5h / 周` 剩余额度展示
- 手动刷新当前状态

## Non-goals
- 云同步
- 团队协作
- 多设备同步
- 历史图表
- 自动切号

## Approach
优先复用 Codex CLI 现有本地认证与本地账户存储，不引入额外账号体系。

## Risks
- 本地数据结构可能随 CLI 版本变化
- 切换行为可能依赖 Codex CLI 进程刷新
- 刚切换后额度数据可能暂时还是旧快照

## Acceptance
- 能查看本地账号列表
- 能切换当前账号
- 能看到当前账号 `5h / 周` 剩余额度
