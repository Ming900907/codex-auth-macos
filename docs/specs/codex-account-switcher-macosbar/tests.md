# Tests

## Normal Cases
- 读取本地账号列表成功
- 识别当前账号成功
- 切换到指定账号成功
- 额度字段可展示

## Error Cases
- 账号文件缺失时给出明确提示
- 额度字段缺失时显示为空态
- 目标账号快照不存在时切换失败并提示

## Edge Cases
- 账号数量为 1
- 当前账号和目标账号相同
- `registry.json` 中账号存在但快照文件缺失

## Regression Cases
- 切换后当前账号标识不丢失
- 刷新后仍能显示最新本地状态
- 当前账号刷新后会把额度快照关联到该账号
- 切换到无新会话账号时可回显该账号最近一次额度快照

## Validation Method
- 为本地解析与切换逻辑补最小单测
- 本地手工运行菜单栏应用验证切换与展示
